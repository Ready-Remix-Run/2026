# Ready · Remix · Run — 2026

**Three classroom-tested CS assignments, presented at the CSTA Annual Conference 2026.**

> Part of the [Ready · Remix · Run](https://github.com/Ready-Remix-Run) archive of classroom-tested CS assignments.

Each assignment below is fully developed, comes with all necessary materials, and includes real implementation experience from the educators who built and taught it.

---

## Assignments

### 🐗 [Truffle Hunt](./truffle-hunt)

Students train a rule-based AI "boar" to help locate hidden truffles in a 10×10 grid — without letting it eat them. By writing pattern-matching rules, students discover how rule-based AI systems make decisions, where they succeed, and where they fall short.

| | |
|---|---|
| **CS Concepts** | Artificial intelligence, rule-based reasoning, pattern matching, logical deduction |
| **Grade Band** | High school · Introductory college |
| **Format** | Browser-based game / individual or paired activity |
| **Time** | 1–2 class periods |
| **Materials** | Computer with a web browser |

---

### 🍕 [Network Roleplay: Pizza Shop](./pizza-shop)

Students become the internet — literally. Playing the roles of clients, routers, and pizza shop servers, they physically pass paper "packets" along yarn "wires" to simulate how data travels across a network. A hands-on introduction to packet switching, net neutrality, and the structure of the internet.

| | |
|---|---|
| **CS Concepts** | Networking, packet switching, routing, net neutrality |
| **Grade Band** | Middle school · High school |
| **Format** | Unplugged / physical roleplay activity |
| **Time** | 1 class period |
| **Materials** | Yarn, scissors, paper, writing implements |

---

### 🖍️ [Crayon Commotion](./crayon-commotion)

Students collaboratively recreate a pixelated image using crayons and a color-coded number system — without knowing what the final image will be. The activity makes concrete how computers store images as tables of numbers, and the big reveal at the end never gets old.

| | |
|---|---|
| **CS Concepts** | Digital representation of images, color tables, GIF format, data encoding |
| **Grade Band** | Elementary · Middle school · High school |
| **Format** | Unplugged / hands-on group activity |
| **Time** | 1 class period |
| **Materials** | Easel pad, crayons, printed mini-pages |

## About This Collection

These three assignments were selected for the inaugural Ready-Remix-Run session at the CSTA Annual Conference 2026. They were chosen because they:

- Teach meaningful CS concepts through engaging, hands-on experiences
- Require minimal or no technology (great for any classroom context)
- Can be adapted for a wide range of grade levels and learners
- Come with complete materials and real classroom implementation stories

See the individual assignment folders for full implementation guides, all materials, and differentiation tips.
